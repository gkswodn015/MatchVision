# MatchVision
YOLO 기반 객체 탐지와 OpenCV 영상 처리를 이용한 축구 경기 분석 프로그램 

## 개요
 
MatchVision은 YOLOv8과 OpenCV를 활용해 축구 경기 영상에서 선수·공을 자동으로 탐지하고, 이동 경로·속도·점유율 등의 분석 데이터를 시각화하는 프로그램입니다. 

---

## 주요 기능

- 선수 및 공 자동 탐지 (YOLOv5/v8 파인튜닝)
- 프레임 간 선수 ID 추적 (ByteTrack / DeepSORT)
- 이동 경로·히트맵 시각화
- 탑뷰(Top-view) 변환으로 포메이션 확인
- 선수별 이동 거리·속도·점유율 리포트 자동 생성

---

## 사용 방법

1. **회원가입 / 로그인** 후 메인 화면으로 이동합니다.
2. **영상 업로드** — 경기 영상 파일을 드래그 앤 드롭하고 경기 이름을 입력해 저장합니다.
3. **팀/선수 등록** — 팀 이름과 선수 정보(번호, 포지션, 이름)를 등록합니다.
4. **영상 분석** — 업로드한 영상을 선택하고 [분석 요청]을 클릭하면 선수·공이 자동 탐지됩니다.
5. **리포트 확인** — 선수별 속도·이동 거리·점유율 통계와 탑뷰 히트맵을 확인합니다. 경기 결과 페이지 URL을 입력하면 득점 기록과 라인업도 함께 불러올 수 있습니다.
